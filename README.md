# Locker-de-Tareas-con-Prueba-de-Auditoria
Es un sistema digital seguro que permite a estudiantes:  Subir tareas con marca de tiempo verificable  Generar comprobantes criptográficos de entrega  Probar autoría y momento exacto de entrega  Mantener integridad de documentos mediante hashing  Crear cadena de custodia auditável para verificaciones futuras
